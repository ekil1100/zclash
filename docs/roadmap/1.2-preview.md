# zclash 1.2 版本规划预览

## 1.2 目标

完善包管理生态，提升配置迁移体验，为更多平台提供原生安装支持。

## 1.2 已完成功能

### 包管理生态扩展 ✅

- **Homebrew 支持** (P17-1A)
  - Formula: `homebrew-zclash/zclash.rb`
  - 使用: `brew tap ekil1100/zclash && brew install zclash`
  - 文档: `docs/install/homebrew.md`

- **Debian 包支持** (P17-1B)
  - 构建脚本: `scripts/build-deb.sh`
  - 产物: `.deb` 包（含 systemd 服务）
  - 文档: `docs/install/debian.md`

- **systemd 服务** (P17-1C)
  - 服务文件: `scripts/zclash.service`
  - 支持: `systemctl enable/start/stop zclash`
  - 文档: `docs/install/systemd.md`

### 安装文档完善 ✅

- Homebrew 安装指南
- Debian/Ubuntu 安装指南
- systemd 使用指南
- curl 一键安装（GitHub raw）

### 迁移规则扩展 ✅ (R28-R29)

- **R28 UNSUPPORTED_PROXY_TYPE_CHECK**: 检测 snell/tuic/hysteria 等不支持代理
- **R29 PORT_CONFLICT_CHECK**: 检测不同代理使用相同端口

**当前规则总数**: R1-R29 (29 条)，全部回归通过
